# Project Tech Stack & Documentation

## Tech Stack

### Frontend
- **React**: A JavaScript library for building user interfaces.
- **TypeScript**: A typed superset of JavaScript that adds safety and better developer tooling.
- **Tailwind CSS**: A utility-first CSS framework for rapid UI styling.
- **Vite**: A fast build tool and development server.
- **Framer Motion**: A library for animations.
- **Lucide React / Material Symbols**: Icon libraries.

### Backend
- **Express**: A minimal and flexible Node.js web application framework.
- **Node.js**: The runtime environment for executing JavaScript on the server.

### Database
- **better-sqlite3**: A fast and simple library for using SQLite databases.

### APIs
- **Google Gemini API (@google/genai)**: For AI-powered features (e.g., generating scenarios, hints).

## Key Concepts Explained

### What is Vite?
**In simple terms:** Vite is your development engine.
Think of it as a super-fast mechanic that keeps your car (the app) running smoothly while you work on it.
- **Instant Updates:** When you save a file, Vite instantly updates the browser so you see changes immediately (Hot Module Replacement).
- **Building:** When you're done, it packs everything up into a small, efficient bundle that can be put on the internet.

### What is React Router?
**In simple terms:** React Router is the GPS for your app.
- **Navigation:** It handles moving between different "pages" (like from Login to Dashboard) without actually reloading the entire webpage.
- **Single Page App (SPA):** It makes your website feel like a native app where transitions are smooth and instant.
